Class ARXFinancialConfiguration

  • java.lang.Object
    • org.deidentifier.arx.ARXFinancialConfiguration

    public class ARXFinancialConfiguration
extends java.lang.Object
implements java.io.Serializable
    Basic configuration of financial variables, such as the publisher's benefit per record or the per-record cost of an attack. Required to analyze re-identification risks on a financial basis, e.g. using a game-theoretic approach.
